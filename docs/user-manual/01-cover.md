# ARKA HERO

## Panduan Pengguna

**User Manual**

<br/>

**ARKA HERO (Human Experience & Resource Optimization)** merupakan aplikasi berbasis web milik PT. Arkananta Apta Pratista, berisi fitur-fitur untuk pengelolaan data kepegawaian, manajemen perjalanan dinas, cuti, rekrutmen, pengelolaan data utama, dashboard monitoring, serta pengelolaan dan penomoran surat resmi secara terintegrasi.
