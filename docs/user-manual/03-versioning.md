# Riwayat versi (Versioning)

| Istilah            | Arti singkat                                                                       |
| :----------------- | :--------------------------------------------------------------------------------- |
| **Version**        | Nomor rilis dokumen (misalnya `1.0.0`); angka bisa mengikuti aturan internal Anda. |
| **Tanggal**        | Kapan perubahan itu diterbitkan atau disepakati.                                   |
| **Rincian update** | Ringkasan apa saja yang berubah (bab baru, perbaikan gambar, koreksi URL, dll.).   |

---

## Tabel riwayat

| Version   | Tanggal                       | Rincian update                                                                                          |
| :-------- | :---------------------------- | :------------------------------------------------------------------------------------------------------ |
| **1.0.0** | _(isi tanggal rilis pertama)_ | Susunan awal: sampul (`01-cover.md`), disclaimer (`02-disclaimer.md`), versioning (`03-versioning.md`). |
